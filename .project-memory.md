- **创建日期**：2026年5月28日
- **重大更新**：2026年6月1日（2.0 版本，新增导航栏、即兴历史模块、60个游戏库、演出准备板块）
- **类型**：即兴戏剧（Improv Theater）教学网站

## 技术架构
- **形态**：纯前端单页应用，零依赖，单文件 `index.html`（约 180KB / 2419 行）
- **路由**：基于 `window.location.hash` 的 SPA 路由
- **数据存储**：学习进度通过 `localStorage` 保存在用户浏览器本地（支持旧 key 自动迁移）
- **设计**：暗色主题，CSS Variables，金色（`#e8b860`）强调色，卡片式布局，sticky 导航栏
- **响应式**：支持移动端（480px / 540px / 740px / 1000px 四个断点）
- **导航栏**：移动端汉堡菜单（纯 CSS checkbox 方案）

## 核心功能
1. **模块文章浏览**：首页 5 张模块卡片 → 点击进入文章列表 → 点击文章进入详情页
2. **学习进度追踪**：每篇文章右侧圆形复选框，进度存于 localStorage
3. **上一篇/下一篇导航**：文章详情页有导航按钮
4. **随机练习生成器**：模块四底部有 22 条练习的随机抽取
5. **游戏库筛选**：模块三按能力标签筛选游戏 + 随机游戏推荐
6. **深链接支持**：URL hash 对应模块或文章 ID
7. **响应式导航栏**：桌面端水平链接，移动端汉堡菜单

## JS 关键函数（位于 index.html 底部 script 标签中）
- `articles` 对象 — 存储 109 篇文章
- `moduleOrder` — 每模块的文章 ID 顺序数组
- `gameSkills` — 游戏能力分类映射
- `showView(id)` — 切换 view 显示
- `navTo(target)` — 导航到模块列表或首页（支持 module0-4）
- `updateNavHighlight(target)` — 更新导航栏高亮
- `openArticle(id)` — 打开文章详情页
- `navArticle(dir)` — 上一篇/下一篇
- `toggleCheck(id)` — 切换学习进度标记
- `loadChecks()` — 从 localStorage 恢复进度（含旧 key 迁移）
- `renderGameList(filter)` — 渲染游戏库列表（支持筛选）
- `randomGame()` — 随机推荐游戏
- `randomPractice()` — 随机抽取练习
- `init()` — 页面初始化

## 后续开发注意事项
- 所有内容在同一个 `index.html` 中，修改文章需在 JS 的 `articles` 对象中操作
- 新增文章需要同时在 HTML 的文章列表区域添加对应的 `<div class="article-row">` 条目
- 新增文章需在 `moduleOrder` 中添加对应的 ID
- 游戏文章需要设置 `skills` 数组用于筛选（对应 `gameSkills` 中的 key）
- 线上部署在 Netlify，修改本地文件后需要重新部署
